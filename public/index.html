SaaShup helps teams focus on product and revenue while the platform handles the SaaS engine around the application. This tool focuses on one operational part of that engine: controlled Docker image upgrades and instance lifecycle actions.

Platform services

The foundation around every instance

CMDB

NetBox keeps infrastructure, Docker hosts and instance inventory organized as the operational source of truth.

Authentication

Keycloak provides identity, SSO and group-based access for the admin and order workflows.

Monitoring

Prometheus collects metrics from the platform, containers and lifecycle automation.

Dashboard

Grafana turns platform metrics into dashboards for capacity, health and operational visibility.

Logging

FlowG centralizes container and system logs so operators can trace activity from one place.

Routing

Traefik routes public traffic to each service and applies the platform edge rules automatically.

DNS and Edge

Cloudflare handles DNS and edge protection for instance domains created by the platform.

Runtime

Docker runs the application containers, volumes and lifecycle operations behind each order.
